  1. SYSTEM LAYERS  (dependency flows downward — upper layers import lower)

  ┌──────────────────────────────────────────────────────────────────────────┐
  │  APPLICATION  (core/)                                                    │
  │   Application · StateMachine · EventBus                                  │
  │   SDL2 + OpenGL context, main loop, state stack                          │
  └──────────────────┬───────────────────────────────────────────────────────┘
                     │ owns / pushes
  ┌──────────────────▼───────────────────────────────────────────────────────┐
  │  GAME STATES  (GameState subclasses)                                     │

             │ reads/mutates             │ calls
  ┌──────────▼──────────┐   ┌───────────▼────────────────────────────────────┐
  │  WORLD DATA         │   │  RENDER  (render/)                             │

  │  no GL, no SDL      │   │   GL 3.3 Core + GLM only — no game logic       │
  └──────────┬──────────┘   └────────────────────────────────────────────────┘
             │ uses
  ┌──────────▼──────────────────────────────────────────────────────────────┐
  │  HEX MATH  (hex/)                                                        │
  │   HexCoord · HexGrid<T>   — pure math, zero external dependencies        │
  └─────────────────────────────────────────────────────────────────────────┘

  StateMachine — stack-based FSM
  ┌──────────────────────────────────────────────────────────┐
  │  stack:  [ WorldBuilderState ]                            │
  │          [ AdventureState   ]  ← top state gets events   │
  │          [ CombatState      ]  ← pushed on dungeon visit  │
  │                                                           │
  │  All states render bottom→top (transparent stack)         │
  │  Only top state runs update()                             │
  └──────────────────────────────────────────────────────────┘

  3. COMBAT SUBSYSTEM  (src/combat/)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Dependency rule: combat/ may import hex/ entities/ world/Resources render/
                   render/ must NOT import combat/

  ┌──────────────────────────────────────────────────────────────────────────┐
  │  CombatMap  (combat/CombatMap.h)   — header-only, pure data             │
  │ ──────────────────────────────────────────────────────────────────────── │
  │  GRID_W = 11 · GRID_H = 5  (flat-top even-q offset hex grid)            │
  │                                                                           │
  │  + toHex(col, row) → HexCoord    even-q offset → axial conversion        │
  │  + fromHex(HexCoord, col, row)   axial → offset  (returns false if OOB)  │
  │  + inBounds(HexCoord)            bounds check                            │
  │  + allHexes()    → vector<HexCoord>   all 55 cells, column-major         │
  │  + playerSpawns() → vector<HexCoord>  col 0, rows ordered 2,1,3,0,4     │
  │  + enemySpawns()  → vector<HexCoord>  col 10, same row order             │
  │                                                                           │
  │  Deployment zones:                                                        │
  │    col 0–1   ░░ player spawn   (blue tint in renderer)                   │
  │    col 2–8      neutral field  (sand)                                    │
  │    col 9–10  ▓▓ enemy spawn    (red tint in renderer)                    │
  └──────────────────────────────────────────────────────────────────────────┘

  ┌──────────────────────────────────────────────────────────────────────────┐
  │  CombatUnit  (combat/CombatUnit.h)                                       │
  │ ──────────────────────────────────────────────────────────────────────── │
  │  - type        : UnitType    species data (by value; → const* when       │
  │                              ResourceManager exists)                     │
  │  - count       : int         creatures remaining in stack                │
  │  - hpLeft      : int         HP of the leading creature                  │
  │  - isPlayer    : bool                                                    │
  │  - isDefending : bool        set by doDefend(); cleared each round       │
  │  - pos         : HexCoord    position on CombatMap grid                  │
  │ ──────────────────────────────────────────────────────────────────────── │
  │  + isDead()    → bool                                                    │
  │  + totalHp()   → int         (count-1)*hitPoints + hpLeft                │
  │  + make(type, count, isPlayer, pos={0,0})  [factory]                     │
  └──────────────────────────────────────────────────────────────────────────┘

  ┌──────────────────────────────────────────────────────────────────────────┐
  │  CombatArmy  (combat/CombatArmy.h)                                       │
  │ ──────────────────────────────────────────────────────────────────────── │
  │  - ownerName  : string                                                   │
  │  - stacks     : vector<CombatUnit>                                       │
  │  - isPlayer   : bool                                                     │
  │ ──────────────────────────────────────────────────────────────────────── │
  │  + allDead()  → bool                                                     │
  └──────────────────────────────────────────────────────────────────────────┘

  ┌──────────────────────────────────────────────────────────────────────────┐
  │  CombatEngine  (combat/CombatEngine.h/.cpp)   — pure logic, no GL/SDL   │
  │ ──────────────────────────────────────────────────────────────────────── │
  │  Owns:                                                                   │
  │    m_player : CombatArmy                                                 │
  │    m_enemy  : CombatArmy                                                 │
  │    m_map    : CombatMap                                                  │
  │    m_queue  : vector<TurnSlot>   initiative order, rebuilt each round    │
  │    m_turn   : int                index into m_queue                      │
  │    m_round  : int                                                        │
  │    m_result : CombatResult                                               │
  │ ──────────────────────────────────────────────────────────────────────── │
  │  Init:  placeArmies() → assigns spawn HexCoords to each stack            │
  │         buildQueue()  → sorts all living stacks by speed desc            │
  │                         (player wins ties)                               │
  │ ──────────────────────────────────────────────────────────────────────── │
  │  Queries:                                                                │
  │    + result() · isOver() · currentTurn() · activeUnit()                  │
  │    + playerArmy() · enemyArmy() · map() · roundNumber()                  │
  │    + reachableTiles()   hexes within activeUnit.moveRange (not blocked)  │
  │    + attackableTiles()  adjacent enemies (melee) / all enemies (ranged)  │
  │ ──────────────────────────────────────────────────────────────────────── │
  │  Actions (stubs — damage resolution in next pass):                       │
  │    + doAttack(targetIndex)   advances turn                               │
  │    + doDefend()              sets isDefending, advances turn             │
  │    + doRetreat()             sets result = Retreated immediately         │
  │    + advance()               moves to next slot; rebuilds queue on       │
  │                              round wrap                                  │
  └──────────────────────────────────────────────────────────────────────────┘

  ┌──────────────────────────────────────────────────────────────────────────┐
  │  CombatRenderer  (combat/CombatRenderer.h/.cpp)   — GL only             │
  │ ──────────────────────────────────────────────────────────────────────── │
  │  Owns:                                                                   │
  │    m_hexRenderer   : HexRenderer     draws the 55-hex battlefield grid   │
  │    m_playerSprite  : SpriteRenderer  billboarded player unit textures    │
  │    m_enemySprite   : SpriteRenderer  billboarded enemy unit textures     │
  │    m_uiVao/Vbo     : GLuint          2D rect for action panel            │
  │    m_uiShader      : Shader          ui.vert / ui.frag                  │
  │ ──────────────────────────────────────────────────────────────────────── │
  │  init():                                                                 │
  │    HexRenderer::init() + loadTerrainTextures("assets")                  │
  │    playerSprite ← assets/textures/units/rider_archer.png                │
  │    enemySprite  ← assets/textures/units/enemy_scout.png                 │
  │ ──────────────────────────────────────────────────────────────────────── │
  │  render(screenW, screenH, engine, cam, hoveredHex, hasHovered):         │
  │    Pass 1 — 3D hex grid                                                  │
  │      For each of 55 hexes:                                               │
  │        tileColor():                                                      │
  │          attackable → red   {0.72, 0.30, 0.22}                          │
  │          reachable  → teal  {0.28, 0.55, 0.52}                          │
  │          player zone (col ≤1) → blue  {0.42, 0.52, 0.70}               │
  │          enemy  zone (col ≥9) → red   {0.70, 0.42, 0.42}               │
  │          neutral              → sand  {0.76, 0.62, 0.38}               │
  │        drawTile with sand texture if loaded, else solid colour          │
  │    Pass 2 — outlines                                                     │
  │        active-unit hex → yellow outline                                  │
  │        hovered hex     → white  outline                                  │
  │    Pass 3 — unit sprites                                                 │
  │        player stacks: draw playerSprite at unit.pos.toWorld()           │
  │        enemy  stacks: draw enemySprite  at unit.pos.toWorld()           │
  │    Pass 4 — 2D action panel (screen-space)                               │
  │        dark bar at bottom 10% of screen                                  │
  │        four coloured buttons: Attack · Defend · Spell · Retreat         │
  └──────────────────────────────────────────────────────────────────────────┘

  CombatState input / camera:
  ┌──────────────────────────────────────────────────────────────────────────┐
  │  Camera2D m_cam  centred on grid (7.5, 4.3), zoom 6.5                   │
  │                                                                           │
  │  Arrow keys / WASD  → pan camera (CAM_SPEED = 8 units/s)                │
  │  Mouse move         → screenToWorld → HexCoord::fromWorld               │
  │                        → m_hoveredHex (white outline if inBounds)        │
  │  Enter / KP_Enter   → engine.doAttack(0)                                │
  │  Space              → engine.doDefend()                                  │
  │  Escape             → engine.doRetreat()                                 │
  └──────────────────────────────────────────────────────────────────────────┘

  TurnSlot  — identifies the active stack in the initiative queue
  ┌──────────────────────────────────────────────────────────────────────────┐
  │  - isPlayer   : bool                                                     │
  │  - stackIndex : int    index into CombatArmy::stacks                    │
  └──────────────────────────────────────────────────────────────────────────┘

  CombatResult enum:  Ongoing · PlayerWon · EnemyWon · Retreated

  4. WORLD DATA OBJECTS  (world/ · hero/ · hex/)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ┌──────────────────────────────────────────────────────────────────────────┐
  │  WorldMap  (world/WorldMap.h)                                            │
  │ ──────────────────────────────────────────────────────────────────────── │
  │  - m_grid    : HexGrid<MapTile>          terrain for every cell          │
  │  - m_objects : vector<MapObjectDef>      towns, dungeons, mines, relics  │
  │  - m_name    : string                                                    │
  │  - m_radius  : int                                                       │
  │ ──────────────────────────────────────────────────────────────────────── │
  │  + tileAt(coord)    + objectAt(coord)    + findPath(from, to)            │
  │  + placeObject()    + removeObjectAt()   + findPathWeighted()            │
  │  + saveJson(path)   + loadJson(path)     + clone()  [explicit deep copy] │
  │                                                                           │
  │  Non-copyable by default. Knows NOTHING about:                           │
  │    render · session state · heroes · visited flags · ownership           │
  └───────────┬───────────────────────────────────────────────────────────── ┘
              │ contains
     ┌─────────┴────────────────────────────────────────────────┐
     │                                                          │
  ┌──▼─────────────────────────────┐    ┌──────────────────────▼────────────┐
  │  MapTile  (world/MapTile.h)    │    │  MapObjectDef  (world/MapObject.h) │
  │ ────────────────────────────── │    │ ─────────────────────────────────  │
  │  - terrain  : Terrain (enum)   │    │  - pos  : HexCoord                │
  │  - passable : bool             │    │  - type : ObjType (enum)           │
  │  - moveCost : float            │    │  - name : string                   │
  │ ────────────────────────────── │    │ ─────────────────────────────────  │
  │  Terrain values:               │    │  ObjType values:                   │
  │    Sand      (moveCost 1.0)    │    │    Town         (pushes CastleState)│

  │    Rock      (moveCost 1.25)   │    │    GoldMine                        │
  │    Oasis     (moveCost 1.0)    │    │    CrystalMine                     │
  │    Ruins     (moveCost 1.25)   │    │    Artifact                        │
  │    Obsidian  (impassable)      │    │                                    │
  │    Mountain  (moveCost 2.0)    │    │  NOTE: visited/owned/looted flags  │
  │    River     (impassable)      │    │  live in AdventureState — NOT here │
  │    Wall      (impassable)      │    └────────────────────────────────────┘
  │    Battle    (triggers combat) │
  └────────────────────────────────┘

  ┌──────────────────────────────────────────────────────────────────────────┐
  │  HexCoord  (hex/HexCoord.h)                                              │
  │ ──────────────────────────────────────────────────────────────────────── │
  │  - q : int   axial column   (world map AND combat grid share this type)  │
  │  - r : int   axial row                                                   │
  │  - s()  →  -(q+r)   [derived, cube constraint]                           │
  │ ──────────────────────────────────────────────────────────────────────── │
  │  + operator +, -, *     + distanceTo()     + neighbor(dir 0..5)          │
  │  + toWorld(hexSize)     + fromWorld(x,z,s)                               │
  │  std::hash<HexCoord>  boost-combine style (non-symmetric)                │
  │                                                                           │
  │  Flat-top orientation. Six directions:  E  NE  NW  W  SW  SE             │
  │     {1,0}  {1,-1}  {0,-1}  {-1,0}  {-1,1}  {0,1}                       │
  └──────────────────────────────────────────────────────────────────────────┘

  ┌──────────────────────────────────────────────────────────────────────────┐
  │  Hero  (hero/Hero.h)                                                     │
  │ ──────────────────────────────────────────────────────────────────────── │
  │  - name      : string                                                    │
  │  - pos       : HexCoord                                                  │
  │  - movesMax  : int  (default 8)                                          │
  │  - movesLeft : int                                                       │
  │ ──────────────────────────────────────────────────────────────────────── │
  │  + resetMoves()   + hasMoves()   + canReach(steps)                       │
  │                                                                           │
  │  PLANNED expansions (separate modules, attach here):                     │
  │    Army<UnitType>  ·  Spellbook  ·  Equipment  ·  SkillTree  ·  XP/Level │
  └──────────────────────────────────────────────────────────────────────────┘

  Spellbook          — set of spell ids the hero knows, mana pool

  Equipment          — 7-slot hero gear (head/neck/body/hands/offhand/feet/ring)
                       applies stat bonuses from Artifact::effects

  CombatEngine damage resolution  — HoMM3 formula + ability tag execution
                       (doAttack stub → real damage, retaliation, death)

  SpellEngine        — interprets Spell::effects JSON at cast time
                       evaluates formula strings against hero spellpower

  TurnManager        — day/week counter, income per mine, weekly unit growth
